🗓️ Daily Planner

A clean and simple daily planner that lets users save events for each hour of the workday. Built with HTML, CSS, JavaScript, jQuery, Day.js, and localStorage.



🔗 Live Demo:

👉 "https://jthapa1987.github.io/third-party-api-workday-scheduler/"   



✨ Features

    🗓️ Shows the current day

    ⏰ Timeblocks for 9AM–5PM

    🎨 Color‑coded for past, present, future

    ✍️ Click any hour to add an event

    💾 Save events to localStorage

    🔄 Events stay saved after refresh

🛠️ Technologies

    HTML

    CSS

    JavaScript

    jQuery

    Day.js

    LocalStorage

⚙️ How It Works

    Timeblocks update based on the current hour

    Colors indicate:

        🩶 Past

        🔴 Present

        🟩 Future

    Events are saved using keys like:

Code

hour-9
hour-10
hour-11
